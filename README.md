# Food Ordering Management System (DOMS)

A comprehensive web-based platform built with PHP and MySQL to manage food menus, customer orders, and administrative tasks. 
The system features separate interfaces for customers and administrators, including a built-in virtual wallet for payments.

**Key Features**

**Customer Features**
Menu Browsing: View available food items with prices.
Order Management: Place new orders and track existing ones.
Virtual Wallet: Pay for orders using a secure virtual wallet system.
Profile Management: Update personal details, contact information, and shipping address. 
Support Tickets: Create and manage support tickets for order assistance.

**Admin Features**
Menu Control: Add, update, or remove items from the food menu.
Order Overview: View and manage all customer orders in real-time.
User Management: Manage customer accounts and permissions.
Ticketing System: Resolve customer support queries through the admin panel.
